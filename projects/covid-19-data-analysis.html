COVID-19 Data Analysis

View on GitHub

Overview

This project presents a Tableau dashboard that visualizes how various global factors — from comorbidities to vaccination rates — affected COVID-19 death rates. The goal was to go beyond basic trends and tell a story using real-world health and policy data.

Problem Statement

COVID-19 created global disruption, but its impact wasn't uniform. I set out to analyze: Which factors influenced mortality most? Did policy response help? How did healthcare systems affect outcomes?

My Approach

  • Collected and cleaned COVID-19 data (Our World in Data)
  • Built calculated fields: deaths per million, diabetes risk bands, policy index
  • Designed multiple Tableau visualizations: scatter plots, heatmaps, dual-axis timelines, and maps
  • Grouped dashboards by theme: health risks, vaccination, policy, hospital strain

Tools & Technologies

Tableau Excel Data Cleaning KPI Design Calculated Fields Conditional Formatting

Key Insights

Health Risk Factors

Countries with high diabetes prevalence had noticeably higher death rates

Vaccination Impact

Vaccination rates correlated with significantly reduced mortality

Policy Effectiveness

Policy stringency helped, but timing and consistency mattered more

Healthcare System Strain

Hospital resource strain led to surges in death rates

Screenshots & Visualizations

Conclusion

This project shows how visual analytics can make complex data accessible. It also highlights how multifactor analysis (not just raw counts) provides clearer decision-making power in public health crises.
